# TimeTrace

TimeTrace is a Chrome extension that helps users monitor their browsing habits, screen time, and productivity directly from the browser. It provides real-time website usage tracking, productivity analysis, and visual insights through an intuitive dashboard.

## Features

- Tracks time spent on websites in real time
- Displays total screen time for the day
- Calculates a productivity score based on browsing activity
- Categorizes browsing into:
  - Productive
  - Neutral
  - Distracting
- Provides a dedicated Today Analytics page
- Shows frequently used websites across tracked days
- Displays daily average screen time and average focus score
- Interactive charts powered by Chart.js
- Clean and responsive user interface

## Screenshots

### Popup
Displays today's screen time, productivity score, category-wise usage, and top websites.

### Dashboard
Provides an overview of daily activity, average usage statistics, focus score insights, and frequently used websites.

### Today Analytics
Offers detailed analytics including website usage charts, website-wise tracked time, productivity score, and category breakdown.
